# 問題 3.14 解答

```scheme
(define (mystery x)
  (define (loop x y)
    (if (null? x)
        y
        (let ((temp (cdr x)))
          (set-cdr! x y)
          (loop temp x))))
  (loop x '()))
```

一般にmysteryは、引数のリストの順番を逆にする。

```scheme
(define v (list 'a 'b 'c 'd))
(define w (mystery v))
```

上の式を評価した後の構造vとwを示す箱とポインタの図

```
                        v--+
                           |
                           V
w -> (* *)->(* *)->(* *)->(* /)
      |      |      |      |
      V      V      V      V
      d      c      b      a
```

```scheme
(print v)
;=> (a)
(print w)
;=> (d c b a)
```
