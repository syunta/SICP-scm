# 問題 1.14 解答

## 木構造

`(cc amount coin)`

```
(cc 11 50) -- (cc -39 50)
    |
(cc 11 25) -- (cc -14 25)
    |
(cc 11 10) -- (cc 1 10) -- (cc -9 10)
    |             |
    |         (cc 1  5) -- (cc -4  5)
    |             |
    |         (cc 1  1) -- (cc  0  1) -- +1
    |
(cc 11  5) -- (cc 6  5) -- (cc  1  5) -- (cc -4 5)
    |             |             |
    |             |        (cc  1  1) -- (cc  0 1) -- +1
    |             |
    |         (cc 6  1) -- (cc  5  1) -- ... -- (cc 0 1) -- +1
    |
(cc 11  1) -- (cc 10 1) -- ... -- (cc 0 1) -- +1
```

## 増加の程度

プロセスが使うスペースの増加は `Θ(n)`。全て1セントで両替するときに最もスペースを必要とする。

プロセスが使うステップ数の増加は指数的に増えている。
